 # Algorithms Table

| Algorithm                            | Type                         | Domain                 | How it Works                                                                | Example Use Case                 |
| ------------------------------------ | ---------------------------- | ---------------------- | --------------------------------------------------------------------------- | -------------------------------- |
| Decision Tree                        | Supervised                   | Tabular Data           | Splits data into branches based on feature values to make predictions       | Loan approval prediction         |
| Random Forest                        | Supervised                   | Tabular Data           | Ensemble of many decision trees that vote for the best prediction           | Fraud detection                  |
| Gradient Boosting (XGBoost)          | Supervised                   | Tabular Data           | Sequential trees improve errors from previous models                        | Customer churn prediction        |
| K-Means Clustering                   | Unsupervised                 | Tabular Data           | Groups data into clusters based on similarity                               | Customer segmentation            |
| Principal Component Analysis (PCA)   | Unsupervised                 | Tabular / Vision       | Reduces dimensionality while preserving variance                            | Data compression & visualization |
| Convolutional Neural Network (CNN)   | Supervised                   | Computer Vision        | Uses convolution filters to detect image features                           | Image classification             |
| ResNet                               | Supervised                   | Computer Vision        | Deep CNN with residual connections to prevent gradient issues               | Medical image analysis           |
| BERT                                 | Supervised / Self-Supervised | NLP                    | Transformer model that understands context in text                          | Sentiment analysis               |
| GPT                                  | Generative                   | NLP / Generative AI    | Transformer-based autoregressive model generating text                      | Chatbots & content generation    |
| GAN (Generative Adversarial Network) | Generative                   | Generative AI / Vision | Two networks (generator + discriminator) compete to generate realistic data | Deepfake images                  |
| Diffusion Models                     | Generative                   | Generative AI          | Gradually remove noise from random data to create images                    | AI art generation                |

# Reflection

Creating this visual framework helped me better understand how different machine learning algorithms relate to specific application domains. Before this assignment, I understood the concepts of supervised and unsupervised learning at a theoretical level, but organizing the algorithms into a structured visual format helped me see how they are applied in real-world AI systems.

One important insight I gained was how different domains rely on different types of algorithms. For example, tabular data problems frequently use models such as decision trees, random forests, and gradient boosting because they perform well with structured datasets. In contrast, computer vision tasks typically rely on convolutional neural networks, which are specifically designed to extract spatial features from images. Natural language processing has been significantly transformed by transformer models such as BERT and GPT, which are capable of understanding and generating human language with remarkable accuracy.

Another key takeaway from this project was the growing importance of generative AI. Technologies such as diffusion models, GANs, and large language models are enabling machines to create entirely new content, including text, images, and even video. Understanding how these models work and where they are used is becoming increasingly important for anyone pursuing a career in artificial intelligence.

Overall, developing this portfolio artifact helped strengthen my understanding of machine learning algorithms while also improving my ability to communicate technical concepts visually. This artifact demonstrates my ability to analyze, organize, and present complex AI concepts in a clear and structured way, which is an important skill for both academic and professional environments.
